Cody

Description

The project is a comprehensive application that helps users determine their home's Energy Performance Certificate (EPC) label based on customizable specifications. Users can input details such as insulation, heating systems, and window types to receive an accurate EPC rating. Built with Node.js for the backend, the app utilizes MongoDB to efficiently store user data and EPC criteria. The front end is crafted using Vue.js, ensuring a dynamic and responsive user experience. For mobile accessibility, the app is developed with Ionic, making it available on both the App Store and as a progressive web app. After evaluating the EPC label, the application provides personalized recommendations for home renovations to improve energy efficiency. This feature helps users understand the potential cost savings and environmental impact of each suggested upgrade. Overall, the project combines modern web and mobile technologies to deliver a seamless and informative user experience.
